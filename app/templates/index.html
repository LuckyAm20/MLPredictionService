Добро пожаловать в ML сервис!

Этот сервис предоставляет возможность загружать изображения и получать предсказания от обученных моделей машинного обучения.

Функциональность:

  • Выбор одной из доступных моделей: ResNet50 или EfficientNet.
  • Предсказание содержимого изображения с использованием мощных нейросетей.
  • История предсказаний для каждого пользователя.
  • Пополнение баланса и просмотр истории транзакций.
  • REST API для интеграции с другими приложениями.
  • Очередь задач через RabbitMQ для обработки изображений.
  • Поддержка работы через веб-интерфейс и API (Postman, cURL).

Как пользоваться?

  1. Зарегистрируйтесь или войдите в свой аккаунт.
  2. Пополните баланс, так как предсказания платные.
  3. Выберите модель для предсказания.
  4. Загрузите изображение и выполните предикт.
  5. Просматривайте историю предсказаний и транзакций.
